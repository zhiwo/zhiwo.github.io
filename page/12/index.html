web.py大文件下载

发表于 2013-11-11   |   分类于 Python

需要下载大文件的时候，如果先将文件直接读入内容再返回，那肯定就很浪费内存，甚至会崩溃。

所以我们需要读一些内容然后直接flush给客户端，但是web.py的文档里面却没有找到flush的方法。

不过在web.py的cookbook中的How to Stream Large Files中看到可以直接yield返回内容。所以，我们可以使用yield来做flush做的事情。

/**

  • 读取Assets文件夹中的图片资源
  • @param context
  • @param fileName 图片名称
  • @return
    */
    public static Bitmap getImageFromAssetsFile(Context context,String fileName)
    {
    Bitmap image=null;
    AssetManager am=context.getResources().getAssets();
    try
    {
    InputStream is=am.open(fileName);
    image=BitmapFactory.decodeStream(is);
    is.close();
    }
    catch(IOException e)
    {
    e.printStackTrace();
    }
    return image;
    }

linux不需要mysql密码连接数据

发表于 2013-10-10   |   分类于 MySQL

redhat:

  1. service mysql stop

  2. service mysql start –skip-grant-tables

  3. mysql -u root -p (Enter 密码为空)

至此已连接上mysql. 推荐你修改密码重新登录，如下继续： (当然你可以创建新用户，具体请自行Google)

  1. update mysql.user set password = PASSWORD(“你的密码”) where user=’root’

  2. exit

  3. service mysql restart

  4. mysql -u root -p (Enter 输入你的密码)
